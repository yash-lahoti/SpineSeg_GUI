# SpineSeg_GUI
Simple GUI for Segmentation


**Creating Environment:**

*Install Pip:*

curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py

python3 get-pip.py

*Install Packages:*

pip3 install argparse opencv-python matplotlib
